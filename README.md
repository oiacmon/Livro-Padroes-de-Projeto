# 📘 README — Projeto Padrões de Projeto (GoF)

Este projeto reúne implementações completas dos **23 padrões de projeto do catálogo GoF**, organizados por categoria e escritos em Java.  
Cada padrão inclui:

- Estrutura de pastas
- Testes JUnit
- Documentação em Markdown
- Diagrama UML (PlantUML)

O objetivo é fornecer um **repositório de referência**, limpo e didático, para estudo, consulta e prática profissional.

---

## 🎯 Objetivo do projeto

O projeto segue os princípios fundamentais descritos no livro *Design Patterns*:

> “Padrões de projeto são soluções reutilizáveis para problemas recorrentes no projeto orientado a objetos…”  
> “Programe para uma interface, não para uma implementação; prefira composição à herança.”
